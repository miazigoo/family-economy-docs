## Описание

Документационный сайт — будущая идея, а не часть текущего MVP scope.

Сайт может появиться после MVP-приложений, когда основные документы будут утверждены и появится необходимость показывать документацию разным аудиториям в удобном виде.

## Возможности будущего сайта

- выбор языка;
- представление для родителей;
- представление для разработчиков;
- представление для инвесторов;
- представление для product-команды;
- публикация только approved docs.

## Границы

Сейчас нельзя создавать код сайта.

Сейчас нельзя перестраивать `docs` под многоязычную структуру.

Сейчас нельзя делать сайт частью MVP.
