## Глоссарий

| English / Source term | Preferred Russian term | Notes |
| --- | --- | --- |
| task | задача | Короткое действие. |
| contract | контракт | Долгосрочная договоренность. |
| habit | привычка | Повторяемое поведение. |
| goal | цель | То, ради чего ребенок копит или выполняет условия. |
| reward | награда | Результат выполнения задачи или контракта. |
| reward obligation | обязательство по награде | Обязательство плательщика после подтверждения результата. |
| proof | подтверждение | Доказательство выполнения. |
| submission | отправка результата | Действие ребенка после выполнения. |
| approval | подтверждение взрослым | Подтверждение результата доверенным взрослым. |
| rejection | отклонение | Отказ принять результат. |
| returned for revision | возвращено на доработку | Результат нужно улучшить и отправить снова. |
| marketplace | рынок / семейный рынок | Только внутри семьи. |
| task marketplace | рынок заданий | Семейный список доступных задач. |
| contract marketplace | рынок контрактов | Семейный список доступных контрактов. |
| treasury | Казначейство | Родительский учет обязательств и выплат. |
| family government | Государство семьи | Опциональный обучающий слой правил. |
| family fund | Семейный фонд | Общий фонд семьи. |
| optional tax | опциональный взнос / налог | Видимое семейное правило, не скрытое списание. |
| charity fund | Благотворительный фонд | Семейная благотворительная цель. |
| exchange | обмен / Биржа | Механизм запроса обмена. |
| exchange request | запрос обмена | Запрос ребенка на обмен ценности. |
| exchange rate | курс обмена | Правило обмена одной ценности на другую. |
| ledger | журнал операций | История начислений, списаний и обменов. |
| balance | баланс | Текущий остаток по журналу операций. |
| payer | плательщик | Тот, кто отвечает за награду. |
| fulfilled | исполнено | Обязательство закрыто. |
| trusted adult | доверенный взрослый | Взрослый, одобренный семьей. |
| unknown adult | неизвестный взрослый | Взрослый вне доверенного семейного контура. |
| character | персонаж | Образ, через который взрослый обращается к ребенку. |
| persona | роль | Альтернативный термин для персонажа. |
| presentation layer | слой представления | То, как действие показывается ребенку. |
| audit | аудит | Проверяемая история действий. |
| safety boundaries | границы безопасности | Продуктовые ограничения безопасности. |
| gambling-like mechanics | механики, похожие на азартные | Запрещенный тип мотивации по умолчанию. |
| public child discovery | публичный поиск детей | Запрещено. |
| public chat | публичный чат | Запрещено для общения с детьми. |
| public market | публичный рынок | Запрещено для детских сценариев. |
| photo report | фотоотчет | Временное визуальное подтверждение. |
| AI helper | ИИ-помощник | Помощник, не финальный судья. |
| manual payment tracking | ручной учет выплат | MVP-модель без банковского API. |
| savings goal | накопительная цель | Цель, к которой направляется награда. |
| Parent Owner | главный родитель / владелец семьи | Внутренняя роль взрослого, управляющего семьей. |
| Child App | Child App / приложение ребенка | Стабильный product label; в тексте можно пояснять по-русски. |
| Parent App | Parent App / приложение родителя | Стабильный product label; в тексте можно пояснять по-русски. |
| visibility | видимость | Для обещаний, условий, обязательств. |
| visible promise | видимое обещание | Центральная идея продукта. |
| family-only | только внутри семьи | Важная граница безопасности. |
| rate-limited | ограничено по частоте | Для напоминаний и похожих действий. |
| manual fulfillment | ручное исполнение | Например, взрослый вручную перевел деньги и отметил выполнение. |
| website | сайт документации | Будущая post-MVP идея. |
